# SOC Analyst Labs Portfolio

This repository documents hands-on cybersecurity labs focused on
Security Operations Center (SOC) fundamentals, endpoint security,
threat detection, and incident response.

## Lab Phases
- Phase 0: Environment Setup & Security Foundations
- Phase 1: Endpoint Security (Microsoft Defender for Endpoint)
- Phase 2: Detection, Investigation & Threat Hunting (in progress)

## Tools Used
- Microsoft Defender for Endpoint
- Microsoft 365 Security Portal
- Azure
- macOS & Windows
- SIEM & Detection Testing
